install uv using command
```bash
pip install uv
```

go to env
```bash
uv venv
```

```bash
source .venv/Scripts/activate
```

install fast-agent-mcp
```bash
uv pip install fast-agent-mcp
```

create agent and configuration
```bash
fast-agent setup
```

run first agent
```bash
uv run agent.py
```

create agent workflow
```bash
fast-agent quickstart workflow
```
